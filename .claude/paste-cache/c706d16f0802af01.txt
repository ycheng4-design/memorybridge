## 1. SYSTEM ARCHITECTURE

### 1.1 Tech Stack (Final Picks ‚Äî No Hedging)

| Layer | Choice | Why |
|-------|--------|-----|
| **Framework** | **CrewAI** (Python) | Role-based multi-agent teams map perfectly to marketing roles (Content Agent, Outreach Agent, Analytics Agent). Production-ready with CrewAI Flows. |
| **Database** | **Supabase (Postgres)** | Real-time subscriptions for dashboard, Row-Level Security for team access, built-in auth, free tier is generous. Airtable is too fragile for production. |
| **Frontend** | **Next.js 15 + Tailwind + shadcn/ui** | Server components, API routes, fast iteration. Dashboard for content queues, outreach queues, analytics. |
| **Queue/Scheduler** | **BullMQ + Redis** | Node.js ecosystem matches Next.js. Handles email scheduling, follow-up chains, content publishing reminders. |
| **LLM Layer** | **Claude API (primary) + OpenAI (fallback)** | Claude for long-form content/analysis, GPT-4o for quick classification tasks. Structured JSON outputs with Pydantic validation. |
| **Email Sending** | **SendGrid** | Best deliverability + analytics. Dedicated IP available. Webhook support for reply tracking. |
| **Lead Enrichment** | **Apollo.io API** | 210M+ contacts, verified emails, 70-80% success rate. Best for founder discovery. |
| **Hosting** | **Vercel (frontend) + Railway/Render (Python agents)** | Vercel for Next.js, Railway for CrewAI Python workers. |
| **Memory/State** | **Markdown files + Supabase** | OpenClaw pattern: persistent context in Markdown. Structured data in Postgres. |

### 2.2 Lead Scoring Model v1 (Rule-Based)

```python
def score_lead(lead: dict) -> int:
    score = 0

    # Stage signals (max 30)
    if lead['stage'] == 'mvp': score += 30
    elif lead['stage'] == 'idea': score += 20
    elif lead['stage'] == 'seed': score += 15

    # Geo signals (max 20)
    if lead['geo'].startswith('US'): score += 20
    elif lead['geo'] in ['UK', 'CA', 'AU']: score += 10

    # Vertical fit (max 15)
    hot_verticals = ['AI', 'SaaS', 'fintech', 'devtools', 'consumer']
    if any(v in lead.get('tags', []) for v in hot_verticals): score += 15

    # Recency signals (max 15)
    if lead.get('recently_funded'): score += 15
    elif lead.get('recently_launched'): score += 10

    # Engagement signals (max 10)
    if lead.get('has_pitch_video_elsewhere'): score += 10
    elif lead.get('active_on_social'): score += 5

    # Source quality (max 10)
    source_scores = {'referral': 10, 'producthunt': 8, 'yc': 8, 'apollo': 5}
    score += source_scores.get(lead['source'], 3)

    return min(score, 100)
```

### 6.3 Email Compliance

- Rate-limit LLM API calls
